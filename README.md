# flutter_sleep

学习flutter的例子项目.  
A Flutter Study Exmaple project.

## 如何运行   
```
请先按照flutter官网[起步](https://flutterchina.club/get-started/install/)搭建好环境,再执行
```
- git clone https://github.com/sleepq123/flutter_sleep.git
- cd flutter_sleep
- flutter doctor
- flutter run (确保已启动虚拟机或者连接真机)
